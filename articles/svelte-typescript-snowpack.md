# Svelte TypeScript Snowpack

My frontend stack for the past 2-3 years has been
React, Redux, and TypeScript.
I've been happy with the user-facing app quality
I've been able to get with this stack
but have wanted to simplify the developer-facing tooling.

Svelte and Snowpack have promise to simplify the stack.
This article is a walkthrough of these two tools together with TypeScript.

```
npx create-snowpack-app svelte-snowpack --template @snowpack/app-template-svelte-typescript
svelte-snowpack
npm start
```
